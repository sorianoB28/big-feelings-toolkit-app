# Check-In Image Assets

This folder structure is reserved for upcoming Canva-based guided check-in image assets.

- `zones/`: regulation zone visuals and related state-based graphics
- `feelings/`: emotion charts, feeling labels, and expression-based assets
- `body-clues/`: body signal illustrations and physical cue graphics
- `breathing/`: breathing exercise visuals, prompts, and sequence images
- `strategies/`: coping strategy cards, regulation supports, and action visuals

Add final exported assets into the matching folders as they are prepared.
